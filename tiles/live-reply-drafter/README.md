# Reply Drafter (live)

Drafts a reply to a **real** Gmail message and **queues the send for your
approval** (draft tier — nothing is sent until you approve it in the Approvals
panel).

Bound to the [`gmail-live`](../../connectors/gmail-live) connector, so it needs
Gmail authorized first (🔌 → ⚙ → Authorize). See
[docs/GMAIL.md](../../docs/GMAIL.md) for the one-time Google setup. For a
zero-setup demo with fake data, use [reply-drafter](../reply-drafter) instead.
